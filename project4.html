Project 4: Integrated Embedded System for Detecting Diabetes Mellitus

I leveraged advanced machine learning and deep learning techniques, including Stochastic Gradient Descent and Neural Networks, to predict diabetes with high accuracy. The model achieved exceptional performance metrics: AUC (97.8%), Classification Accuracy (96.7%), F1-score (96.3%), Precision (96.6%), and Recall (96.7%).

This project demonstrated the effectiveness of combining robust algorithms with cutting-edge neural network architectures to deliver reliable and high-performing predictive models for healthcare applications.
